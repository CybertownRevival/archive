GUIDELINES FOR 3D WORLDS IN COLONY CITY

Worlds for Colony City are created using VRML as a programming language. Worlds should be created using the latest version of VRML called VRML 2.0.

Many Tools/Programs exist to create these worlds.

The basic guidelines are:

  • The file size of the world should be under 150K including textures. Your world should be compressed (gzipped) to make the file size smaller.
  • The content of the worlds should adhere to the City Constitution and rules (such as no pornographic images, etc...).
  • There is a World Builders Guild existing in Colony City headed by the City Architect. If you need advice or help, contact them.
