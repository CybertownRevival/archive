2.1. How to use "My Book"

Give your personal book a title and a brief description of what it's about. The last five people who've looked at the book are listed on the title page. The title page also includes information about how often the book has been read and when it was last changed.

Click on the 3D object to edit your book. After you've finished editing your information, save the changes by clicking on "Update".

Use the "Next page" button to go to the next page.
Use the "Previous page" button to go back a page.
You can add a page with the "New page" button; you can add up to 20 pages.
Use the "Delete page" button to remove the current page.
To edit the current page, use the "Edit page" button.
