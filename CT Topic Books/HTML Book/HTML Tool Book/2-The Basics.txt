<font color=white size=3><P>*Format*</P></font><br></center>
<font color=coral>All HTML documents should follow the same basic format - otherwise some commands may not
 function properly. The basic format for all HTML documents should be:</font></p>
<font color=yellow><P><center>
< html ><br>
< head ><br>
< title ><br>
< /title ><br>
< /head ><br>
< body ><br>
< /body ><br>
< /html ></center></p><br></font>
<P></P><font color=skyblue>Head: </font>
<font color=coral>Head tags surround the title and the introductory text.</font></p>
<P></P><font color=skyblue>Title: </font>
<font color=coral>This should describe the content of the page. It should be descriptive, but less than 64 
characters. Bookmarks pick up on your page title. An example of a poor title would be "Cats". A better 
title would be "Long-haired Cats and Kittens".</font></p>
<P></P><font color=skyblue>Body: </font>
<font color=coral>The main part of your page goes between the</font>
<font color=yellow>< body > & < /body ></font>
<font color=coral> tags.</font></p>
<P></P><font color =skyblue>Closing tags: </font>
<font color=coral>You must remember your closing tags. In some browsers, the document will still function 
without some of them. With others, it will not.</font></p></td>
