<P>Font Size<br><font color=coral>
There are two ways to change your font size. SPECIFY THE ACTUAL FONT SIZE <font color=yellow>< font size="n" > <font color=coral>where "n" is a number from 1 to 7.
Remember to always close with the<font color=yellow> < /font > <font color=coral>tag.<br>
<font color=purple>
<Font size="1">Hi<font>
<Font size="2">Hi</font>
<Font size="3">Hi</font>
<Font size="4">Hi</font>
<Font size="5">Hi</font>
<Font size="6">Hi</font>
<Font size="7">Hi</font></p><br>
<font color=skyblue><font size=3>
<P>Relative Font Size<br><font color=coral>Specifying the relative font size.<font color=yellow> < font size="+n" > or < font size="-n" >.<font color=coral>
Basically, you are specifying how many sizes larger or how many sizes smaller than the preset font size.<br>
<font color=pink>
<Font size="-2">Bye</font><br>
<Font size="-1">Bye</font><br>
<Font size="+1">Bye</font><br>
<Font size="+2">Bye</font><br>
<Font size="+3">Bye</font><br>
<Font size="+4">Bye</font><br><br>
