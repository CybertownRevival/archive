<b>Lesson #1 Line Break, Paragraph & Center</b>
<hr>
 <XMP><BR> = Line Break</XMP> 
Use this tag at the end of a line to make a line break.<br> 
This moves the text to the next line in your message.<p>
<XMP><P> = Paragraph</XMP><br>
Use this tag to make a new paragraph or <br> 
put a space between lines.<p>
<XMP><center> = To center your typing.</XMP>
<XMP><center>Text goes here</center></XMP>
Write a message you want centered between the <br>
open center and the closed /center.<p>
NOTE: Line Break and Paragraph code don't need <br>
to be closed. But you can close them if you want.<br> 
All other code once opened must always be /closed.<p>
