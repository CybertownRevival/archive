<font size="2" color="#0066FF" face="verdana"><b>
A sometimes forgotten attribute when creating<br>
a clickable image is the <u>border="0"</u> attribute.<br>
When this is not added to the image code, a<br> 
"highlight box" will form around the clickable image.<p>
Computer users may see this "highlight box" but<br>
don't know how easy it is to make it disappear.</b></font>
<p>
<FONT SIZE="2" COLOR="#c71585" face="verdana,helvetica,sans-serif"><b>
Here is a screen capture of this "highlight box" <br>
around a clickable image when <u>border="0"</u> is not used.<br>
Note the code also, lacking the <u>border="0"</u> attribute.
</b></font>
<p><img src="http://dannydking.org/images/vine.jpg" width="260" height="59" border="1" alt="without border equals zero example"><br>

<FONT SIZE="2" COLOR="#0066ff" face="verdana,helvetica,sans-serif"><b>

add the <u>border="0"</u> attribute to the image code.
</b></font>
