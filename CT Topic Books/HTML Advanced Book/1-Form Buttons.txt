</b></font>

Copy & Paste The Code:
</b></font>

<input type="submit" value="Click Me"><br>
</form>

Internet Explorer users might have noticed the cool transitions comimg to this Html Extra's page. This code is very simple to do. Just place this meta code between the <head> & </head> tags.
</b></font>
<p>
<font size="2" color="#c71585" face="verdana"><b>
<META http-equiv="Page-Enter"<br>
content="revealTrans(Duration=#,Transition=#)">
</b></font>
<p>
<font size="2" color="#0066ff"  face="verdana"><b>
You will need to change two parts of this code. First, you will need to change the # after Duration to the number of seconds that you want the transition to take. Then, you will need to change the # after Transition to the number of the type of transition that you want.
<p>
<a href="http://www.cybertown.com/cgi-bin/cybertown/iwos?ac=next&ID=o0c69a063&PAGE=4">
Click here for a list of the transitions and their corresponding number value.</a>

<font size="2" color="#0066FF" face="verdana"><b>
This is an Internet Explorer 5.0+ code, it will not work with Netscape.
<p>
Ever save a web page and see an icon displayed in your favorites ?
<p> 
This can be achieved by first creating a 16X16 icon, you can look for icon software at <a href="http://www.download.com/" target="_window">Download.Com</a>. 
After you have created your icon, name it whatever you would like, just be sure to have the .ico extension.
Then upload your icon to the same directory that the web page is in.
<p>
The final step is to add the code below between the <head> and </head> tags of your html document.
</b></font>
<p>
<font size="2" color="#c71585" face="verdana"><b>
<link rel="shortcut icon" href="http://whatever.com/myicon.ico"> 
</b></font>
<p>
<font size="2" face="verdana" color="#0066ff"><b>
This will also display an icon in the URL address bar once the web page has been saved in your favorites.
<p>
For more on the favorite icon and an icon editor visit <a href="http://www.favicon.com/" target="_winow">Favicon.Com</a> 
</b></font>
