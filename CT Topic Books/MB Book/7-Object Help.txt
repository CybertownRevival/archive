<strong><center>VRML File Sizes</center></strong>
<ul><li>The maximum size for the (Gzipped) VRML file is 55 kB.</li>
<li>You may only upload one texture with your item. File size must be less than 40kb.</li>
<li>You may include up to 2 Universal Media textures in your object. When using Universal Media, a maximum of three textures (in total) may be used and total file size (textures + item, but not thumbnail) must not exceed 150 KB.</li>
<li>All thumbnails must be 450x450 or smaller in image size and smaller than 40kb in file size.</li></ul>
